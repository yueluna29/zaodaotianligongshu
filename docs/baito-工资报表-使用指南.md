# baito 老师 · 工资报表 使用指南

每月一次的工时提交流程，大概 3 步就能搞定。

---

## 1. 日常：上完课录入当天工时

1. 登录系统，左边菜单点 **工资报表**
2. 日历上点你要登记的日期（默认周视图，也可以切月视图）
3. 右边「选中日期」区点右上的 **+** 加一笔
4. 填好这几项：
   - **工种**（EJU講師 / 大学院理工 / 学部等，决定时薪）
   - **开始 / 结束时间**
   - **学生姓名**（请填全名）
   - **课程 / 工作内容**
   - 当天有交通费就填，没有就留 0
5. 点顶部 **💾 保存** 按钮（多笔改动一起保存）

> 日期上有小圆点的说明那天已经有记录。

### 其他当日报销
右侧「其他报销」卡片点 **+**：金额 + 说明（文具、资料费等）。

### 签单提成（按月累计，不按日）
如果你有签单任务，右侧「签单提成」卡片点 **+** 添加每单：学生姓名 / 学费 / 提成率。系统自动算金额。

---

## 2. 备注（可选）

月薪资总览下方有一块「备注」：这个月需要说明的情况写这里（请假、换班、特殊报销等）。

写完点右上**绿色勾勾 ✓** 保存。没改动时勾勾是灰色的。

---

## 3. 月底：提交本月报表

月底确认当月所有记录完整后：

1. 点顶部 **📨 提交本月报表** 按钮
2. 弹窗里核对：
   - 总工时 / 课时费 / 交通费 / 其他报销 / 签单提成 / 总合计
3. 上传 **两张打卡照片**（必填）
   - 点虚线框选图片上传，会自动压缩到 500KB 以内
   - 上传成功后会显示缩略图
   - 想换就点缩略图再选一张
4. 两张都传好后，**确认提交** 按钮才会亮起
5. 点「确认提交」完成

> 提交后当月工时会被**锁定不可修改**，需要改的话联系管理员解锁。

---

## 常见问题

**忘记登记某天，隔天能补吗？**  
能，直接在日历上回到那天 + 添加，保存就行。

**保存了没？**  
改了但没点 💾 时，顶部会显示「未保存变更」。保存完按钮变灰。

**两张打卡照片传哪张？**  
通常是出勤 / 退勤各一张，按老师习惯来。

**照片上传失败**  
换一张重试；或者检查网络。还不行联系管理员看 edge function 日志。

**已提交的月份想改**  
联系管理员解锁，解锁后可以重新编辑并重新提交。

**日历跳到其他月份**  
顶部月份切换 + 日历上方也可以选，选中日期前后还有 ← → 切当天。
