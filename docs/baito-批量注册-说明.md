# Baito 老师批量注册 CSV 模板说明

模板：`baito-批量注册-模板.csv`

## 列说明

| 列 | 必填 | 规则 |
|---|---|---|
| `login_id` | ✅ | 4-20 位英文字母/数字，不能有汉字/空格。会自动转小写。老师登录用 |
| `姓名` | ✅ | 可以汉字、假名，随便写，就是档案里显示的名字 |
| `公司` | ✅ | 必须是下面 4 个之一：`世家学舍` / `紫陽花教育` / `早稻大连` / `早理金华` |
| `部门` | 可空 | baito 可选：`大学院` / `学部` / `文书` / `语言类`。不填就留空 |
| `事务时薪` | 可空 | 円（整数），不填这个工种就留空 |
| `讲师时薪` | 可空 | 対応 DB 里的 `専業課老師` |
| `答疑做题时薪` | 可空 | 対応 `答疑做題` |
| `研究计划书时薪` | 可空 | 対応 `研究計画書修改` |
| `过去问时薪` | 可空 | 対応 `過去問` |
| `EJU班课时薪` | 可空 | 対応 `EJU講師（班課）` |

## 固定值（不用填，脚本自动设）

- **密码**：`123456`（老师登录后自己改）
- **雇佣类型**：`アルバイト`
- **入职日期**：执行当天

## 提交方式

把填好的 CSV **内容**贴到对话里（或者拖文件过来），我解析后直接批量创建账号 + 写时薪表。每一行的执行结果（✓ / ×）会报给你看。

## 注意

- 一行一位老师
- 用半角逗号分隔
- `login_id` 重复会被跳过（包括之前已注册的）
- 时薪留空不会创建 pay_rates 行；想给那个工种设 0 円可以填 `0`
- 如果 CSV 打开乱码，保存时选 **UTF-8** 编码
