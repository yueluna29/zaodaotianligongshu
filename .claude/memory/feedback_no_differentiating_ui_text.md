---
name: 共用 UI 不写"部分人可选"类提示
description: 规则差异私下沟通，不在共享界面写"这部分人可以跳过/那部分人必须填"
type: feedback
---

在所有用户看得到的公共页面（提交对话框、表单提示等），不要写"线下老师请传 X，远程老师可跳过"这类**按角色差异化**的文案。

**Why**：如果规则在界面上写死了"某些人可以不做"，那些按规矩做的老师看到会觉得不公平（"凭什么别人可以不传"）。即使 admin 真的有合理区分（确实一部分老师不用传），也不要放在公共 UI 里暴露。

**How to apply**：
- 硬性拦截要删就彻底删（不再 disabled、不再红色警告）
- 配套的解释文案也全部移除，不要留"可选"/"选填"/"视情况"之类
- 规则由 admin 入社时**私下跟每位老师一对一说明**；界面上只显示中性的操作选项（有上传入口，但不说"谁该传谁不用传"）
- 当前案例：月度报表提交的打卡照片——功能保留但完全去掉提示文案
